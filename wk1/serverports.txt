Q1- What is the definition of a server port?
A1- A port number is a way to identify a specific process to which an internet or other network message
	is to be forwarded when it arrives at a server.

Q2- What is the server port range?
A2- Port numbers range from 0 to 65535, but only port numbers 0 to 1023 are
	reserved for privileged services and designated as well-known ports.

Q3- How many bits does the number represent?
A3- 0000 0000 0000 0000 - 1111 1111 1111 1111
